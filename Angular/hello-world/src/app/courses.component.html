Using *ngFor

  • {{ course }}
This list of courses is propagated using an Angular directive called *ngFor, which is used like a for loop.
courses.component.html
<ul> <li *ngFor="let course of courses"> { { code } } </li> </ul>

Attribute Binding

To bind attributes to values from your .ts file use attr.theAttributeName="valueFromTypeScriptFile"
courses.component.html
<table> <tr> <td [attr.colspan]="colSpan"></td> </tr> </table>

Event Binding



courses.component.html
<button (click)="onSave($event)">With Event Binding</button>
courses.component.ts
onSave($event) { $event.stopPropagation(); //This is used to prevent the bubbling up of events from the DOM. console.log("Button was clicked", $event); alert("Button was clicked"); }

Event Handling

Explicitly grabbing the keyup event as an input and handling the event



courses.component.html
<input (keyup)="onKeyUp($event)"/>
courses.component.ts
onKeyUp($event) { if ($event.keyCode === 13) alert("ENTER was pressed"); //The key code for ENTER is 13 }

Event Filtering

Grabbing the event directly as an option of keyup

courses.component.html
<input (keyup.enter)="onKeyUpWithEventFiltering($event)"/>
courses.component.ts
onKeyUpWithEventFiltering($event) { alert("ENTER was pressed"); //The key code for ENTER is 13 }

Template Variables



courses.component.html

courses.component.ts
onKeyUpUsingVariable(email) { alert(email); }
